# Estado del Proyecto

## Situación Actual

El frontend React está alineado con la API Spring Boot de producción y consume el prefijo `/api/v1`.

## Módulos Validados

- Autenticación
- Condominios e infraestructura
- Residentes y personas
- Tickets e incidencias
- Comunicados

## Integración

- Web, móvil y escritorio consumen la misma API.
- La base de datos de producción es PostgreSQL.
- Las respuestas de la API siguen el formato `ApiResponse`.
